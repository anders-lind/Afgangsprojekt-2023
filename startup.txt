Pre-requisites:
sudo apt install ros-noetic-aruco-detect (only once)


Start roscore:
roscore

Launch Aruco-Detect launch file (new terminal):
roslaunch aruco_detect aruco_detect.launch dictionary:=0 fiducial_len:=0.2 publish_images:=true

Launch Obstacle-detector launch file (new terminal)
roslaunch obstacle_detector obstacle_detector.launch

Start Rosbag (new terminal in folder with bagfiles):
rosbag play -l <insert_bag_file> /camera_floor/driver/color/camera_info:=/raspicam_node/camera_info /camera_floor/driver/color/image_raw/compressed:=/raspicam_node/image/compressed
